2장 텍스트 관련 태그

2.1 h태그

  • 강조를 주고 싶은 글에 사용하며, 보통 제목을 쓸 때 사용한다.
  • h1 부터 h6까지 사용가능하고 숫자가 작을 수록 글씨가 커진다.
  • h태그는 block 요소이다.
  • 기본적으로 margin값이 설정되어 있다.
  • 폰트 크기는 2em, 1.5em, 1.17em, 1em, 0.83em, 0.67em 으로 되어 있다. (2018.09.26확인)

2.2 p태그

  • 문단 입력 태그이다.
  • p태그는 block 요소이다.
  • 기본적으로 margin값이 설정되어 있다.
<p>텍스트</p>

텍스트

2.3 span태그

  • 줄바꿈 없이 영역을 묶을 때 사용한다.
  • span 태그는 inline 요소이다.
<span>내용</span>
내용

2.4 br태그

  • 줄 바꾸기 태그이다.
  • 다음과 같이 사용한다.
  • 닫는 태그가 없다.
텍스트<br>다음텍스트
텍스트
다음텍스트

2.5 hr태그

  • 수평 줄을 넣을 때 사용하는 태그이다.
  • p태그는 block 요소이다.
  • 닫는 태그가 없다.
이 글 밑에 수평줄<hr>
이 글 밑에 수평줄

2.6 bloackqoute태그와 q태그

  • 인용글을 넣을 때 사용하는 태그이다.
  • 자동으로 줄바꿈과 들여쓰기가 된다.
  • q 태그를 사용해서 인용글을 넣을 수도 있는데, q 태그는 줄바꿈이 없고 따옴표가 입력이 된다.
<blockquote>인용 내용</blockquote>
<q>인용 내용</q>
인용 내용
인용 내용

2.7 pre태그

  • 입력한 그대로 화면에 표시하고 싶을 때 사용한다.
  • pre 태그를 사용하면 스크린리더기에서 해당 부분을 skip한다.
<pre>텍스트</pre>

텍스트

2.8 code태그

  • 소스코드를 넣을 때 사용한다.
  • 보통 pre태그와 함께 사용한다.
<pre><code> function test() {...}
</code></pre>

 function test() {...}

2.9 strong태그와 b태그

  • 글씨를 굵게 표시할 때 사용한다.
  • strong태그와 b태그는 inline요소이다.
  • 두 태그는 화면상으로 동일하게 표현되지만, strong 태그로 표기할 경우 스크린리더기에서 강조 텍스트로 구분해준다.
<strong>굵게 강조할 텍스트</strong>
<b>굵게 강조할 텍스트</b>
굵게 강조할 텍스트 굵게 강조할 텍스트

2.10 em태그와 i태그

  • 이탤릭체로 표시할 때 사용한다.
  • em태그와 i태그는 inline요소이다.
  • strong 태그와 동일하게 em 태그를 사용하면 스크린리더기에서 강조 텍스트로 구분해준다.
<em>이탤릭체</em>
<i>이탤릭체</i>
이탤릭체 이탤릭체

2.11 s태그

  • 취소선을 넣을 때 사용한다.
<s>취소선</s>
취소선

2.12 u태그

  • 밑줄을 넣을 때 사용한다.
<u>밑줄</u>
밑줄

2.13 small태그

  • 글자를 작게 표시할 때 사용한다.
<p>내용<small>(첨부확인)</small></p>

내용(첨부확인)

2.14 mark 태그

  • 형광펜 효과를 낼 때 사용한다.
<mark>텍스트</mark>
텍스트

2.15 ruby 태그

  • 동아시아 국가들의 글자를 표시할 때 사용한다.
  • 주석으로 표시할 내용을 태그 안에서 태그로 표시한다.
<ruby>내용<rt>주석</rt></ruby>
내용주석

2.16 sub태그와 sup태그

  • 아래 첨자와 위 첨자를 넣을 때 사용한다.
<p>log<sub>2</sub>10</p>
<p>2<sup>10</sup></p>
