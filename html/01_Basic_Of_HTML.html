1장 HTML의 기본

1.1 HTML이란?

  • HyperText Markup Language
  • 쉽게 말해 웹에서 자유롭게 오갈 수 있는 웹 문서이다.
  • 팀 버너스리에 의해 만들어진다.

1.2 팀 버너스 리

  • 웹 창시자
  • W3C(World Wide Web Consortium) 설립

1.3 Markup 이란?

  • 태그를 사용해 문서에서 어느 부분이 제목이고 본문인지, 어느 부분이 사진이고 링크인지 표시하는 것

1.4 웹 브라우저의 종류

  • chrome(google)
  • mozila(firefox)
  • Internet explorer(MS)
  • Edge(MS)

1.5 HTML에서 사용되는 기본 태그

1.5.1 DOCTYPE 태그

  • HTML문서의 가장 첫 줄에 쓰여진다.
  • 현재 문서가 HTML언어로 작성된 웹 문서라는 뜻이다.
  • 다음과 같이 작성한다.
        <!DOCTYPE html>

1.5.2 html 태그

  • 웹 문서의 시작과 끝을 나타내는 태그이다.
  • lang이라는 속성을 사용해 문서에서 사용할 언어를 지정할 수 있다. (국가별 언어코드 사용)
  • 언어를 지정하는 이유는 검색이나 화면낭독기에서 해당언어를 참조할 수 있다.
  • 다음과 같이 작성한다.
<html lang="ko"> 

</html>

1.5.3 head 태그

  • 웹 브라우저가 웹 문서를 해석하기 위해 필요한 정보들을 입력하는 부분이다.
  • title 태그를 사용해 문서제목을 브라우저 창에 표시할 수 있다. (즐겨찾기로 추가할 때도 동일)
  • style태그와 link 태그 등이 사용되는 곳이다.
  • meta 태그를 이용해 문자 인코딩 및 문서 키워드를 지정하고, 간단한 설명에 대해 지정가능하다.
<head>
    <title>여기에 title을 적는다.</title>
    <meta name="description" content="여기에 사이트에 대한 짧은 설명을 입력한다">
    <style> ... </style>
    <link> ... </link>
</head>

1.5.4 body 태그

  • 실제로 웹 브라우저 화면에 나타낼 내용을 입력한다.
